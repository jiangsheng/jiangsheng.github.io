很多学生、开发者和媒体出席

现在在由微软全球副总裁、前亚洲研究院院长张亚勤在主讲移动计算

目前讲到的

* 桌面电脑有4亿，但是芯片有四百亿
* 可移动计算是通讯平台，也是移动的中心
* 移动通讯正在转型，类似于电视转型一样，从语音传输转到数据传输
* 统一的通讯网络
* 用户需求：移动办公、移动娱乐、移动财务管理。
* 无缝连接。智能终端。
* 语音是最直接、最自然的交互。语音识别、控制、合成。
