   |image|

   在消失4个版本之后，类向导终于重返Visual
   C++。新的功能：搜索现在可以部分匹配而不是从字符串开始匹配。

   测试版还是存在一些问题，向导不是总能找到现存的函数，以致删除函数功能不是总有效。在打开很多文档的时候尝试打开Class
   Wizard会出现“value does not fall in expected range”
   错误。再就是性能问题，打开的文档越多，类向导启动所需的时间就越长。
