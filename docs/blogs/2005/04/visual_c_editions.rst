Visual C++ 2005的版本区别
=========================================

.. post:: 21, Apr, 2005
   :tags: MFC
   :category: Visual C++, Microsoft Foundation Classes
   :author: me
   :nocomments:

经常看见有些人问Visual C++ 2005里面为什么没有了MFC。实际上，MFC只是在目前免费下载测试版的Express版本里面没有，在其它的版本里面都有——但是一般都是要付钱的。具体的各个版本的比较可以参见参考部分的链接。不过对于用VC来学习.Net开发的程序员来说，Express版就足够了。
