.. _srw4_pilot_emma_sheen_commentBegin:

艾玛在上一次大战中作为隐藏要素，本次则一开始就加入。机动战士系的好机体不够用，她也就轮流上上超级高达混等级（仅限真实系），直到需要压级等地图武器练级为止。在这之前，因为其机体太好被他人抢走，她大概会坐上回避能力欠缺因此需要机师弥补的刚坦克。

地形适应很差，在真实系机师中是拖后腿的宇宙B，平时换机时要注意地形适应。好在有地图武器之后刷级不是问题，而她的精神不错，有觉醒可以调整地图武器发射角度，50级就可以用三次激励。回避能力在非新人类中排名非常高。缺点是二次行动比较慢。

第四次S中改善了回避。シールド防御从1级提升到6级。

花园丽尽管初期能力不强，但是成长率出人意料地高，特别是远程攻击能力。而且有幸运，精神和特技都学得很快。对空适应是机动战士系少见的A，可以开Zガンダム赚钱。

地形适应是空A陆B，适合バイアラン、Sガンダム和ExSガンダム。第四次中リ・ガズィ是空B，花园丽这个空A去开有点浪费了，第四次S中リ・ガズィ变成了空A所以也是个选项。

第四次S中シールド防御从1级提升到4级。

.. _srw4_pilot_hathaway_noah_commentBegin:

布莱德之子，老爸老妈在三个月之前的上一次机器人大战才结婚，三个月之后的本次机器人大战中理论还在老妈肚子里，就已经成年加入我军的哈撒维……因为母亲ミライ・ヤシマ的关系，和阿姆罗有些联系。

在开局的速路线\ :ref:`隐藏要素 <srw4_missable>`\ 的 \ :doc:`../walkthrough/05a_bright_s_return`\ 之后和尼莫一起加入。选择缓路线则会得到凯拉、克莉丝和巴尼，同时还能获得最强芯片FATIMA。

虽然是有觉醒新，但是能力较凯拉差，凯拉和巴尼的机体还关乎罗莎米娅的加入条件，而且即使是在第四次S中也会强制废弃他的尼莫，所以如果不是要看完全剧情或者收集图鉴的话，没有什么必要加入。

虽然可以培养出新人类能力，但是本作中装备浮游炮的盟军机比较少，他的地形适应和远攻击也很差，所以用处不大。

有个真实系少见的気合，比较适合F91，但是如果用地图炮刷了级的话，应该不缺激励了。另外，他是陆A宇B，F91是陆B宇A，浪费了双方的能力。

第四次S中增加了远攻击和回避。增加的切り払い可以升到5级。
