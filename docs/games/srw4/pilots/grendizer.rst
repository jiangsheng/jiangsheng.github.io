グレンダイザー登场人物
=========================================

---------------------
我方人物
---------------------

^^^^^^^^^^^^^^^^^^^
デューク=フリード
^^^^^^^^^^^^^^^^^^^

杜克是魔神系蹭经验主力。因为大家都轮流跟着古连泰沙蹭经验，以至于デューク=フリード的等级会被堆得很高。后期会和牧場ひかる分到A队，其他魔神系到B队，以至于其他人要开始自力更生了。

具备主力所需全部精神，副座可专心辅助。至于其攻击力稍逊，因为等级会被刷得很高，并非很大问题。

第四次S中地形适应现在是全A，陆战和海战不再是短板。另外增加了命中、回避和技量。

^^^^^^^^^^^^^^^^^^^
牧場ひかる
^^^^^^^^^^^^^^^^^^^
牧場光和杜克一起加入，基本都是和古连泰沙合体出战，所以自己的能力差也不是问题。二次行动学得比较慢。后期会和杜克分到A队，建议分队之前换乘到ドリルスペイザー上去，这样A队就多了个无消耗的输出。

再動和復活都是非常珍贵的精神，而且出得非常早。唯二的不费力就能习得复活的人员，另一个是希拉。激励则比较烂大街，可以用地图武器批量制造。说老实话，精神不怎么适合作为杜克的辅助，其他候选都有幸运或者热血。不过她的精神和能力太不适合独立作战了，如果杜克要带别人练级的话，是否派出场是个难题。

少见的海A，但是本作海战太少了，マリンスペイザー没有用武之地。宇宙适应是B，不过后面宇宙战的时候,魔神系大多分到B队去了，平时在A队还是可以跟杜克一起合体作战。

^^^^^^^^^^^^^^^^^^^
マリア＝フリード
^^^^^^^^^^^^^^^^^^^
玛丽亚是杜克的妹妹。很早就开着ドリルスペイザー加入我军。

不像在第三次大放光彩的时候，本作能力比较差，自己的机体大招是近战的，她的近攻击却是本系列倒数第二，只比牧場ひかる要强。技量非常低，不要指望暴击率。

精神方面，出场比较早，有热血和幸运，比较适合当杜克的辅助。最后还能学会再动和补给，适合给地图炮机体当辅助。

地形适应方面，空B陆A匹配能乘换的大部分机体。宇A被浪费掉了，没有宇A的机体可以换乘。

后期分到B队，不能再蹭古连泰沙，开始需要自力更生，而且机体大概也会被牧場ひかる抢走，可以用魔神Z练级。不过因为近攻击太低，用ブレストファイヤー练级还是很勉强。考虑到A队地图难度问题，大魔神也会被铁也开去A队，
后期大概还是开维纳斯A或者戴安娜A作为修理和地图武器支援人员。
