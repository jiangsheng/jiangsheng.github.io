.. _srw4_pilot_maria_fleed_commentBegin:

玛丽亚・弗里德是\ :ref:`デューク・フリード <srw4_pilot_duke_fleed>`\ （杜克・弗里德）的妹妹。在\ :ref:`第十四话 空飛ぶマジンガーZ <srw4_walkthrough_14_mazinger_z_flying_in_the_sky>`\ 中开着\ :ref:`ドリルスパイザー <srw4_unit_drill_spider>`\ （钻地斯陪沙）加入我军。

不像在第三次大放光彩的时候，本作能力比较差，自己的机体大招是近战的，她的近攻击却是本系列倒数第二，只比\ :ref:`牧場ひかる <srw4_pilot_hikaru_makiba>`\ （牧场光）要强。技量非常低，不要指望暴击率。

精神方面，出场比较早，有热血和幸运，比较适合当杜克的辅助。最后还能学会再动和补给，适合给地图炮机体当辅助。

地形适应方面，空B陆A匹配能乘换的大部分机体。第四次中宇A被浪费掉了，没有宇A的机体可以换乘。第四次S中各个斯陪沙的宇宙适应提到了A。后期分到B队，不能再蹭\ :ref:`グレンダイザー <srw4_unit_grendizer>`\ (古连泰沙)，开始需要自力更生的时候，可以用\ :ref:`マジンガーZ <srw4_unit_mazinger_z>`\ (魔神Z)练级。不过因为近攻击太低，用胸部火焰练级还是很勉强。考虑到A队地图难度问题，\ :ref:`グレートマジンガー <srw4_unit_great_mazinger>`\ (大魔神)也会被铁也开去A队，
后期大概还是开\ :ref:`ビューナスA <srw4_unit_venus_a>`\ （维纳斯A）或者\ :ref:`ダイアナンA <srw4_unit_diana_a>`\ (戴安娜A)作为修理和地图武器支援人员。

.. _srw4_pilot_duke_fleed_commentBegin:

杜克是弗里德星球的王子，在维加联盟的袭击后被迫逃往地球。被宇宙科学研究所所长宇门源藏收留并收养，改名为“宇門大介”，在研究所和宇门的熟人牧场丹兵卫(牧叶光的父亲)经营的白桦牧场帮忙。在本作中，出于未知原因，他加入了AEUG（奥古），与\ :ref:`クワトロ・バジーナ <srw4_pilot_quattro_bajeena>`\ (库瓦特罗·巴吉纳) 一同作为增援，在中盘的宇宙路线的\ :ref:`第二十四话Ａ エゥーゴとの接触 <srw4_walkthrough_24a_encounter_with_the_aeug>`\ 中加入朗德·贝尔。

魔神系蹭经验主力。因为大家都轮流跟着\ :ref:`グレンダイザー <srw4_unit_grendizer>`\ (古连泰沙)合体蹭\ :ref:`经验值 <srw4_exp>`\ ，以至于等级会被堆得很高。后期会和\ :ref:`牧場ひかる <srw4_pilot_hikaru_makiba>`\ (牧场光) 分到A队，其他魔神系大多到B队，以至于很多人要开始自力更生了。

作为少数拥有宇宙A级适应力的超级系驾驶员，也能在后期保持战力。具备主力所需全部精神，副座可专心辅助。至于其攻击力稍逊，因为等级会被刷得很高，以至于暴击率不低，并非很大问题。

第四次S中地形适应现在是全A，陆战和海战不再是短板。另外增加了命中、回避和技量。虽属主角级角色却未收录语音，缘由在于配音演员富山敬当时正住院治疗。

.. _srw4_pilot_hikaru_makiba_commentBegin:

原作名牧葉ひかる，牧場ひかる是笔误。

牧場光和\ :ref:`デューク・フリード <srw4_pilot_duke_fleed>`\ (杜克·弗里德) 一起加入，基本都是和\ :ref:`グレンダイザー <srw4_unit_grendizer>`\ (古连泰沙)合体出战，所以自己的能力差也不是问题。二次行动学得比较慢。后期会和杜克分到A队，建议分队之前换乘到\ :ref:`ドリルスパイザー <srw4_unit_drill_spider>`\ (钻地斯陪沙)上去，这样A队就多了个无消耗的输出。

再動和復活都是非常珍贵的精神，而且出得非常早。唯二的不费力就能习得复活的人员，另一个是\ :ref:`シーラ・ラパーナ <srw4_pilot_ciela_lapana>`\ (谢拉·拉帕纳) 。激励则比较烂大街，可以用地图武器批量制造。说老实话，精神不怎么适合作为杜克的辅助，其他候选都有幸运或者热血。不过她的精神和能力太不适合独立作战了，如果杜克要带别人练级的话，是否派出场是个难题。

少见的海A，但是本作海战太少了，\ :ref:`マリンスペイザー <srw4_unit_marine_spacer>`\ (潜水斯陪沙)没有用武之地。宇宙适应是B，不过后面宇宙战的时候,魔神系大多分到B队去了，平时在A队还是可以跟杜克一起合体作战。
