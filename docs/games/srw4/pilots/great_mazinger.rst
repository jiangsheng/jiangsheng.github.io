.. _srw4_pilot_jun_hono_commentBegin:

炎纯作为修理机机师来说能力是正常地差。地形适应也和自己的机体不符合，\ :ref:`ビューナスA <srw4_unit_venus_a>`\ （维纳斯A）是空B陆A，她是空A陆B，也不适合本系大把的空B陆A的机体，后期\ :ref:`グレンダイザー <srw4_pilots_grendizer>`\ （古连泰沙）会分在A队，而玛丽亚会去B队，所以你大概会想让她提前去开\ :ref:`ダブルスペイザー <srw4_unit_double_spacer>`\ (飞行斯陪沙)继续蹭等级。

精神方面虽然比较偏战斗向，但是培养她也并无坏处，42级可以学会激励。因为SP充沛，是很早能使用3次激励的机师之一。不过\ :ref:`シーブック・アノー <srw4_pilot_seabook_arno>`\ (西布克，仅限第四次)和\ :ref:`ファ・ユイリィ <srw4_pilot_fa_yuiry>`\ （花园丽）也可以……

没有幸运是个大问题，地面适应也是B，不适合同系大部分机体。作为空中A级热血拥有者，更适合战斗，但是本作用地图炮刷级之后可大批培养战斗要员，以至于辅助机师更为稀缺，出场率低于\ :ref:`弓さやか <srw4_pilot_sayaka_yumi>`\ (弓沙也加) 。如果不想用地图炮刷级的话，因为SP充沛，可以每回合加热血，还是可以考虑让她作为战斗机师。

.. _srw4_pilot_tetsuya_tsurugi_commentBegin:

开局第一话，\ :ref:`兜甲児 <srw4_pilot_koji_kabuto>`\ 就说很担心铁也，因为他失踪了，后面发现他的机体\ :ref:`グレートマジンガー <srw4_unit_great_mazinger>`\ (大魔神)都被抢走了，还好随着剧情发展，他被证实安然无恙。

魔神系唯一会加速的。由于\ :ref:`ボスボロット <srw4_unit_boss_borot>`\ (波士机器人)的机动力和射程问题，大概是这个机体的最佳驾驶员。没有ひらめき，后期打BOSS比较乏力。但是魔神系本来就是用来清理小兵的坦克单位，此缺陷并不致命。因为\ :ref:`ダンバイン <srw4_pilots_dunbine>`\ (丹拜因)系会分去B队，后期会在经历游戏最难的几话的A队中和\ :ref:`デューク・フリード <srw4_pilot_duke_fleed>`\ (杜克·弗里德) 一起作为防御中坚。

如果你想听点不同的台词，那可以在第四次S中让铁也和甲児交换机体。
