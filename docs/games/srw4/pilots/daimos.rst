.. _srw4_pilot_kazuya_ryuzaki_commentBegin:

和エリカ的恋情是剧情发展主线之一。

开局之后第一个加入的超级系，在超级系中能力可算顶尖，加上近身武器大多会加暴击率，即使热血很慢才学得，攻击力也不能算差。

要说缺点的话，空中适应是B是个大问题，机体在第四次S中升级到空A也没有效果，开战时仍要落地。宇宙适应是A则是被机体拖累了。机体的大招无法在宇宙中使用（明明原作中能正常施展）。

第四次S中加入了语音，除了命中和直感之外的能力也全面上升。近攻击值和万丈并列，技量更是上调到我军超级系最高。仍旧有不适应空战和宇宙战的问题。

虽然精神设定方面十分符合其火爆脾气，但性格不知为何被设定为弱気。能力值和经验值偏低。
