版本之间的变化
==============

第四次的ROM有两个版本，1.0 (CRC32:BE764D4C) 和1.1 (CRC32 : 63883E1E)。最大的区别是1.0中最高等级为100(查看时花屏)，地图武器攻击友方机体有经验值，以及一次最多能获得65535经验值。在1.1版本中最高等级为99，地图武器攻击友方机体没有经验值，一次最多能获得32767经验值。另外，初版的ドム在机器人大百科中登场作品为機動戦士ガンダム0080，次版修正。

第四次和第四次S也并非完全相同。下面列出PS版本修改的一些地方：

* 部分人物有了语音。
* 地图上芯片位置变化。
* 有些单位不再被强制离队（\ :ref:`ダンクーガ <srw4_unit_dancouga>`\ 、\ :ref:`コンバトラーV <srw4_unit_combattler_v>`\ 、\ :ref:`NT-1アレックス <srw4_unit_nt_1_alex>`\ 、\ :ref:`GM III <srw4_unit_gm_iii>`\ 、\ :ref:`ガンタンク <srw4_unit_gun_tank>`\ 、:ref:`エルガイム <srw4_units_heavy_metal_l_gaim>`、\ :ref:`カルバリーテンプル <srw4_unit_calvary_temple>`\ ）。\ :ref:`シュウ・シラカワ <srw4_pilot_shu_shirakawa>`\ (白河愁) 加入时也不会发生离队事件。
* NPC可以自己控制。
* 即使说得了经过修改的敌方单位加入，武器的改造段数也会被初始化到0。
* 巨蟹座AB型现在可以学到两个热血……

---------------
剧情
---------------

* 开始的时候增加了一话，名为“接触”。总回合数要求也相应变化。参见\ \ :ref:`隐藏要素 <srw4_missable>` \。
* \ :doc:`../walkthrough/39a_ryune_capriccio_gato`\或者\ :doc:`../walkthrough/39b_ryune_capriccio_gilliam`\ 之后，如果说得了リューネ＝ゾルダーク，下面一话进入新增的一话\ :doc:`../walkthrough/39c_rescue`\ 。
* \ :doc:`../walkthrough/33a_total_balance`\ 或者\ :doc:`../walkthrough/33b_beyond_the_todd`\ 进入战斗之前\ :ref:`ダンクーガ <srw4_unit_dancouga>`\ 小队或者\ :ref:`コンバトラーV <srw4_unit_combattler_v>`\ 小队可以不用离队。


新增:ref:`三将军事件 <srw4_missable_guest_generals>`。他们将能生存到游戏结束。

---------------
能力
---------------

* 主人公的生日血型精神对应表改变，能力上升。超级系主人公机的装甲上升。
* \ :ref:`ファンネリア・アム <srw4_pilot_fanaria_amm>`\ (范妮莉亚·阿姆) 的精神熱血的习得等级从31改为7。\ :ref:`浪花十三 <srw4_pilot_juzo_naniwa>`\ 的精神偵察的习得等级从40改为35。\ :ref:`ブライト・ノア <srw4_pilot_bright_noah>`\ (布莱德·诺亚) 的精神必中的习得等级从24改为14。
* 机师的切り払い和シールド防御等级全面上升
* オーラバトラー的オーラ斬改为消耗EN20，ハイパーオーラ斬り改为消耗EN35
* \ :ref:`ライディーン <srw4_unit_reideen>`\ 的武器ゴッドボイス 威力从5900增至7200，消耗EN从130减至110
* ダンクーガ队机师藤原忍的精神加速(15级)变成魂(45级)。ダンクーガ增加空中移动能力（机器人大百科也随之更新），运动性和装甲上升，断空剣和断空砲威力大幅上升，EN消耗减少。
* \ :ref:`シーブック・アノー <srw4_pilot_seabook_arno>`\ (西布克·阿诺) 的精神“激励”改成気合
* \ :ref:`マジンガーZ <srw4_units_mazinger_z>`\ 队（甲児、鉄也、デューク、マリア、ジュン、さやか、ボス、ひかる）的机体的装甲有所强化
* \ :ref:`エルガイム <srw4_units_heavy_metal_l_gaim>`\ 系的パワーランチャー的威力強化、弾数減少
* 火星の決戦中シュウ＝シラカワ所属部队的等级不再是99，我方气力成为80。
* 第四次中原本有改造的机体入手时武器都没有改造。\例如 :ref:`GP-02A <srw4_unit_gp_02a>`\ 的核弹。
* \ :ref:`サイバスター <srw4_unit_cybuster>`\ (塞巴斯塔)的コスモノヴァ的威力从4080提升至4800。 \ :ref:`ヴァルシオーネR <srw4_unit_valsione_r>`\  (瓦尔西奥妮R)武器威力上升。
* \ :ref:`F-91 <srw4_unit_f_91>`\ 的ヴェスバー的威力从3300提升至3800，消耗EN从90减至50，必要气力改为100。运动性和装甲上升。
* \ :ref:`νガンダム <srw4_unit_nu_gundam>`\ (ν高达)的フィンファンネル、\ :ref:`ノルス・レイ <srw4_unit_nors_ray>`\ （诺尔斯·改）的ブラスナックル威力大幅提升
* \ :ref:`GP-03デンドロビウム <srw4_unit_gp_03_dendrobium>`\  (GP-03 D)的大型ビームサーベル和メガビーム砲的威力大幅提升
* \ :ref:`ウィーゾル改 <srw4_unit_weesol_kai>`\  (维佐尔·改)的エレメンタルフュージョン的威力从2900增至3900
* \ :ref:`コンバトラーV <srw4_unit_combattler_v>`\  (孔巴特拉V)的グランダッシャー的威力从5100增至5800 
* \ :ref:`グランゾン <srw4_unit_granzon>`\  (古兰森)和\ :ref:`ネオ·グランゾン <srw4_unit_neo_granzon>`\ (真·古兰森)\ 的ングラビトロカノン的威力从1540增至2500，縮退砲必要气力取消。
* \ :ref:`ボスボロット <srw4_unit_boss_borot>`\  (波士机器人)的スペシャルボロットパンチ和スペシャルデラックスボロットパンチ的威力大幅上升
* \ :ref:`Ζガンダム <srw4_unit_z_gundam>`\ (Z高达)攻击力上升
* \ :ref:`百式 <srw4_unit_hyaku_shiki>`\  运动性上升
* \ :ref:`グレートマジンガー <srw4_unit_great_mazinger>`\  (大魔神)的ブレストバーン的威力大幅上升 \ :ref:`マジンガーZ <srw4_unit_mazinger_z>`\  (魔神Z)和\ :ref:`グレンダイザー <srw4_unit_grendizer>`\ (古连泰沙)武器威力上升
* \ :ref:`ビギナ·ギナ <srw4_unit_vigna_ghina>`\ (比基纳·基纳)装甲上升
* \ :ref:`ビルバイン <srw4_unit_bilvine>`\  (比尔拜因)装甲下降
* \ :ref:`ゴーショーグン <srw4_unit_goshogun>`\  (战国魔神)、 \ :ref:`ダイモス <srw4_unit_daimos>`\  (戴莫斯)、\ :ref:`ザンボット3 <srw4_unit_zambot_3>`\  (赞波特3)武器威力上升
* \ :ref:`真·ゲッター2 <srw4_unit_shin_getter_2>`\ (真·盖塔2)的ドリルテンペスト的射程从1~3变成了1~7。ｹﾞｯﾀｰﾐｻｲﾙ取消。
* 其他的一些机师和机体能力的修改
