精神
===========
.. _srw4_mechanics_sprit_command:

相当于其他SLG或者RPG的魔法。每个机师都具有不超过6个精神，精神在达到一定等级的时候悟得。

相对于其他精神，加速和幸运对机师表现影响巨大。在机体移动力普遍为9以下时，+5移动力带来显著差距，在有有利\ :ref:`地形 <srw4_terrain>`\ 、有高奖金敌人会提前撤退或者需要护卫特定NPC时尤为如此。幸运则是将获得\ :ref:`经验值 <srw4_exp>`\ 翻倍，因为本作后期敌人等级提高较速，缺乏此精神的机师将在和Boss作战时受到等级压制。

常被低估的激怒实为游戏核心要素，挑衅机制将使得敌人在同一回合开始行动而非预设的次第行动，善用走位可使得敌人摆出有利于地图武器攻击的阵型。另外一些敌人受到伤害时撤退，若无望将其击落获得经验和资金，则将其驱离更加有利。注意部分Boss会免疫挑衅效果。本作具有这个精神的有ボス、忍、大作、主角和恋人、主角和恋人的精神通常会选择有奇迹的，忍和大作则需要承担战斗职责，故需多多培养ボス。ボス命中能力低下，应换乘其他机体来保证能获得经验值。如果敌人众多，多派出几个会激怒的，在削弱敌人方面可能比地图武器更有效。

本作最实用的精神为奇迹，效果为ド根性+気合×2+加速+幸運+必中+ひらめき+魂，消耗仅40SP。唯持有此精神的机师甚少，游戏中必定有此精神的只有チャム，然需56级方可习得，除和白河愁作战的最终话之外，敌方等级最高仅50级，以至于需要用地图武器刷级方可。对于主人公则可以通过指定生日和血型获得。9月2日O型正因如此被粉丝铭记。

本作中SP恢复的唯一方式为复活，因此应尽量多培养能复活的机师。游戏机制可被滥用，两个单位可复活彼此，使得SP无限。

后期因为限界达到上限255的原因，应善用かく乱、集中和地形效果。部署时应索索有利地形，优先具有恢复和防御、防御修正的基地、城市、建筑、森林、山地、瓦砾、陨石带和殖民地。

游戏地图中散布着资金或道具，可用探索精神获取其地址，然后派遣\ :ref:`メタス (MA) <srw4_unit_methuss_ma>`\ 、\ :ref:`GP-01Fb <srw4_unit_gp_01fb>`\ 等探宝机前往。如果有耐心，可以探索之前存盘，获得地点之后读盘再前往。侦察也可以以类似的方式使用。

关于每个精神有哪些持有者，参考\ :ref:`第四次机师数据<srw4_pilots_data_snes>`\ 和\ :ref:`第四次机师数据<srw4_pilots_data_ps>`。

精神修改
----------------------------

.. _srw4_mechanics_sprit_command_cheat:

SP消耗的地址可以通过查找连续的根性、集中、補給、再動等精神的SP消耗获得。精神的顺序可以参考前面的表格。

主角的特殊诞生日精神和习得等级以如下顺序存储：

* 9/2 O型
* 12/24 B型
* 6/25 AB型
* 4/29 A型
* 3/13 B型
* 4/11 O型
* 8/12 A型
* 11/16 AB型
* 1/29 B型
* 2/9 A型 

第四次的生日血型数据每组12个字节，含六个精神，每个精神2字节。低字节为精神代码，高字节为习得等级。第四次的生日血型数据每组14个字节，除了六个精神外，还多了两个字节00 00。


除了以上特殊诞生日和血型之外，主角的普通诞生日精神以如下顺序存储：

* 白羊座 （依次为O,A,B,AB四组，下同）
* 金牛座
* 双子座
* 巨蟹座
* 狮子座
* 处女座
* 天秤座
* 天蝎座
* 射手座
* 山羊座
* 水瓶座
* 双鱼座

主角和恋人的默认精神和习得等级参考\ :ref:`序幕<srw4_walkthrough_preclude>`\ 。
