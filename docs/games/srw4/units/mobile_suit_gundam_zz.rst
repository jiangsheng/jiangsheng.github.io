机动战士ZZ高达登场机体
========================================


------------------
ＧＭⅢ
------------------
第四次里有强制废弃事件，不好改造。第四次S里可以一用，陆地适应加强到了A，比ガンダムmkII要强。

------------------
ガルスＪ
------------------
除了陆地的适应性是A之外，其他的能力都不怎么样。

----------
ズサ
----------
完全的杂鱼机。

--------------------
ケンプファー
--------------------
很奇怪地武器全体对海适应都是A，但是机体本身的海中适应是C，只适合在岸边对付海中的敌人。在\ :doc:`../walkthrough/06a_stampede`\ 这一话中会作为NPC援军登场。
