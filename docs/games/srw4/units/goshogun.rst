.. _srw4_unit_goshogun_commentBegin:

虽然主题歌里面说宇宙Space Number One，但是宇宙地形适应只有B的战国魔神……

罕见的远程型超级机器人，主力武器太空火箭筒和必杀技豪闪光均为远程攻击，移动后缺乏攻击力，只能拿光剑砍人。加入较晚，且战斗精神集中于真吾，使用有所不便。唯加入时近战人员过剩，可与之互补。抛去必杀技不谈，仅以太空火箭筒一项，即可媲美\ :ref:`Ζガンダム <srw4_unit_z_gundam>`\ (Z高达)的攻击力，唯命中略微不足。在习得魂之后更可用来在近距离不足以解决Boss时用豪闪光远程补伤害。

.. _srw4_unit_god_neros_commentBegin:

原作最终话中，由多库加的ゼニガスキー・ジッター（泽尼加斯基·吉塔）正在制作的35体合体机器人。未能投入实战，所以武器是原创的。35台机甲合体而成的终极机器人，但由于合体场景过长而被删减。

在第三十九话リューネ・カプリッチオ中作为增援登场。虽然HP较高，但可能是设定错误，明明是空陆两栖机体却没有空中适应性，火力薄弱到连战斗机都能击落它。不过由于驾驶员\ :ref:`超人工知能 <srw4_pilot_super_ai>`\ 属于空中A级，实际战斗力相当于空中C级。

若在援军登场前通关，则不会收录至机器人大图鉴。

\ :ref:`アトールV <srw4_unit_a_taul_v>`\  (亚托鲁V)没有单独能力设定，其数据直接指向本机的地址。
