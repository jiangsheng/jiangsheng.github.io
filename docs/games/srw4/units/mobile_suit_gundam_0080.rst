特洛伊木马号是地球联邦军建造的天马级两栖攻击舰的第五艘。代替在上一次大战中已达到极限的白色基地作为朗德·贝尔的第一艘母舰。最初驾驶员是托雷斯，后来是布莱德。由于反复的艰苦战斗而没有适当的补给和维护，最终遭遇了与白色基地相同的命运。

改造一路继承，可以一直用到最后一话（但是你可能想在最后一话派出奥拉系主舰）。有自残练级计划的可以考虑先不改造机体性能，在机动战士系和重战机系的机师全体刷满之后再开始改造。没有自残练级计划的话可以正常改造，但是优先级仍然是地图武器最高，早点刷出两动装载大部队抢占有利地形用，战力不能指望布莱德。

后期圣战士战舰入手之后因为支持精神更好会优先出击，但是后期分两队的情况下有几话会强制出战，还是有表现的机会的。

在第10话之后被アーガマ号替换。注意不要在正常离队之前把它修改掉，不然会在\ :doc:`../walkthrough/31_storm_of_kilimanjaro`\ 这一话导致游戏无限循环。

^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
ザク改
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
取代前作的扎古在本作起登场。作为DC阵营的战力出现，但作为敌人仅在最初几个剧本中登场。

相较于《EX》之前的扎古，虽然增加了手榴弹和冲锋拳套等武装，但机体颜色不统一，动力管颜色差异明显，武器图形更是全部采用普通扎古的，完全混淆了扎古与扎古改的设计。考虑到高达系列在本作发售时资料已相当丰富，资料不足的可能性微乎其微，这显然是Winky原创的扎古，仅冠以"扎古改"之名。

弹药量和攻击力均偏低，作为MS属于最弱性能。根据路线不同可作为巴尼的座机获得，但性能仅可以作为真男人的挑战考虑。

游戏中段存在与杰刚选择性废弃的事件，舍弃此机可获得2000资金。也可用于罗莎米娅的出击事件，但该机仍会随之消失。罗莎米娅的其他潜在搭档机体还有杰刚与刚坦克，但为获取真实路线限定的Ex-S高达与雅克多加，必须牺牲其中一台。若想同时招募罗莎米娅，则需额外保留杰刚或扎古改其中一台。换言之，若想在本作中保留扎古改，只能放弃上述任一机体加入（真有人这么做么），或选择超级路线。

第四次S中MS普遍有所改善，但是本机没有。由于GM III与刚坦克的废弃事件被移除，在罗莎米娅的剧情线更容易保留巴尼机体至最终战。但杰刚的双机选择废弃事件依然存在，因此最终可保留的机体上限为雅克多加在内的三机中的两架。
