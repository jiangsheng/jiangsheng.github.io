=================================

-------------
Ｓガンダム
-------------
.. _srw4_unit_s_gundam:

看起来很强大，但是武器多而不精，残弹数量太少，改造价值不大。因为不需要新人类能力的原因，可以让コウ去坐。后续作品中量産型νガンダム和ドーベンウルフ也可以让旧人类开，但是本作中旧人类的好机体不多。

-------------
ExSガンダム

.. _srw4_unit_ex_s_gundam:
