ガンダムセンチネル登场机体
=================================

因为版权纠纷，本作中只有机体登场，本作之后再无出场。

-------------
Ｓガンダム
-------------
.. _srw4_unit_s_gundam:

正式名称为「卓越高达（Superior Gundam）」，开发代号为「ι（伊奥塔）高达」。

第四次中，走真实系路线的话，与现有刚坦克二选一即可获得。若刚坦克已被废弃则可直接获取，因此选择事先废弃为划算。第四次S中则无二选一事件，不用废弃。因为可以升级到Ex-S所以限定了同一话说得的罗莎米娅的入手路线。

兼具高机动性和变形能力，账面数据强大，但是地形适应为B，武器多而不精，残弹数量太少。因为不需要新人类能力的原因，可以让コウ去坐直到GP03入手。后续作品中量産型νガンダム和ドーベンウルフ也可以让旧人类开，但是本作中旧人类的好机体不多。

光束智能枪虽威力偏低但拥有8射程的优秀性能，升级为Ex-S后射程却莫名降至7。而光束加农炮虽然射程只有7，在Ex-S形态下攻击力提升，若进行改造建议优先升级光束加农炮。

MS形态下与科配合默契，但考虑到G巡洋舰的运用，也可搭载西布克、花等空战适应性高的驾驶员。

第四次S中虽然不再需要和刚坦克二选一，但是包括地形适应性在内的性能均未和其他机体一样进行改善，以至于实用性降低。

---------------
Gクルーザー
---------------
原作S高达无法变形，Ex化本质是为了实现变形功能而进行的改造，如果S高达原本就可以变形，那Ex化的目的到底是什么呢。

跑路形态。对空适应是A，空A的机师可能用这个形态战力更强。但是本作空A级MS驾驶员极为稀缺。

G巡洋舰需在地图上变形一次或在通关画面变形检查单位性能，否则无法登记至机器人大图鉴。若未登记就直接升级为Ex-S形态，将需要重新游戏才可以登记到图鉴。

-------------
ExSガンダム
-------------
.. _srw4_unit_ex_s_gundam:

名称读作"伊克塞斯高达"。正式名称为"Extrordinary-Superior GUNDAM（非凡卓越高达）"。在S高达上加装了后述用于变形为G巡洋舰的追加部件的机体。旨在解决S高达变形时的强度不足问题，并全面强化攻防能力。虽然机体因此大型化，但通过在关键部位追加装甲、增设推进剂与推进器（随之将背包替换为大功率推进单元）、搭载I力场发生器及反射式通信装置等措施，整体性能得到显著提升。因为Ex化是为了实现变形能力，所以MS形态是偶然产物。

作为搭载旧人类的机体，其性能堪称顶尖，但陆地地形适应性依然仅为B级，如果不是GP-03D的地形适应过于拉跨，可能在科的座驾上有得一拼。在罗莎米娅路线入手，所以要放弃说得フォウ。罗莎米娅可能驾驶的刚坦克与S高达存在排他性，要同时达成真实系独占的Ex-S高达和雅克多加，以及收罗莎米娅，必须在8回合以上完成"发端"任务以获取杰刚与扎古改。

继承S高达的改造。新增的大招残弹数量仍然太少，改造价值不大。要入手的话，需要放弃フォウ，但是既然都收了Ｓガンダム，为什么不把它升级呢？フォウ可以在走超级系路线的时候收。

第四次S中虽然不再需要和刚坦克二选一，但是包括地形适应性在内的性能均未和其他机体一样进行改善，以至于实用性降低。

-------------------
Gクルーザー (EXS)
-------------------
原作中Ex-S是以G巡洋舰为主要形态，但是本作中G巡洋舰形态反而性能降低，主要用来跑路。对空适应是A，空A的机师可能用这个形态战力更强。

G巡洋舰需在地图上变形一次或在通关画面变形检查单位性能，否则无法登记至机器人大图鉴。
