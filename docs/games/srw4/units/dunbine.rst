ダンバイン登场机体
=====================

本作起奥拉战士需圣战士技能才能使用奥拉斩，但作为补偿被设定为零消耗武器，虽入手初期略显冷门，经改造后却能展现违规级强度。第四次S中奥拉斩改为EN消耗武器。

---------------------
ダンバイン
---------------------

由于比尔拜因正在维修，翔便驾驶了这架丹拜因出场。虽然终于得以作为主角机登场，但机体性能早已跟不上翔的能力，更糟的是无需等待比尔拜因，仅过几话就能获得隐藏机体塞拜因或兹瓦士，而且获取时还必须使用本机的灵能转换器导致其报废，可谓待遇堪忧。虽具备可用性能，但因前述因素往往很快便被弃用。若坚持使用可改造，否则装备芯片直到换机就好。

第四次超级机器人大战S的开场动画中首次采用3D模型登场。奥拉斩改为能量消耗式。

---------------------
ボチューン
---------------------
波臣作为玛贝露、、尼、基恩的机体登场，但在获得比尔拜因时会强制废弃1架，若满足利姆鲁加入条件则需用莱尼克交换再强制废弃1架。若选择加拉利亚加入路线，则仅能获得玛维尔的1架机体，因此可在获取比尔拜因前自行废弃变现；但需注意若选择加拉利亚未加入路线，自行废弃会导致部分飞行员无机可用。

该机体不仅射程上限为1，本作中装甲值降至400，连敌人杂兵都不如，在性能与定位上均遭受冷遇。

第四次S中奥拉斩改为消耗EN。同时取消了与莱尼克强制二选一的设定，仅需以1架机体作为与比尔拜因交换的代价。

---------------------
ズワウス
---------------------

兹瓦士和シルキー同时加入。和サーバイン、ダンバイン三选一。出于某种问题，オーラソード和クラフトドリル的工艺钻共享改造段数。


运动性略逊于塞拜因，但HP较高且装甲更厚，更有HP恢复（小），在后期回避到达限界255时生存能力属于我军最强。然而于比尔拜因相比，缺乏远程能力，移动力亦处于劣势。强化部件方面，相较机动性提升系，装甲系反而更契合其特性，搭载拥有铁壁的加拉利亚亦是良策。但需注意HP过高以至于敌人经常会绕过它，不适应团队作战。


第四次S中奥拉斩改为消耗EN。此外因比尔班装甲被下调，本机成为奥拉战士中防御力最高的机体。至于优先选择机动性出众的塞拜因还是装甲坚固的兹瓦士，可根据玩家偏好自由抉择。

---------------------
サーバイン
---------------------
塞拜因和シルキー同时加入。和ズワウス、ダンバイン三选一。只能进行近战，但是威力强大。


虽装甲较兹瓦士稍薄，但在奥拉战士更需要的能力——运动性与移动力方面占据优势。然而本作限界上限仅255，后期生存能力略逊于兹瓦乌斯。不过两机均具备战斗至终局的能力，单纯根据个人喜好选择亦无不可。

相较于比尔拜因，这台机体更极致地体现了奥拉巴托拉核心价值——"高机动性"。虽无射击武器，但本作敌方单位的AI设定是命中率低下时会主动切换至巴尔干炮这类命中修正高的近战武器。因此只要让回避率高的翔驾驶员坐镇前线，敌方单位就会主动靠近。这使得射程的缺陷几乎不构成实质影响。不过远距离的劣势并非完全丧失，如果敌人完全缺乏近战能力，若不想陷入进退维谷的窘境，建议启用比尔拜因。

若让翔驾驶本机，其尚可的射击攻击力将沦为废属性，因此相较之下，格斗能力与翔相当而射击偏低的加拉利亚反而更契合。

获取条件需将翔移动至特定地图的特定位置，但游戏中毫无提示（因附近埋藏强化零件，探索时可大致推测位置），且该点远离部队展开点，更与敌军部队方向完全相反，不知情者基本无法获取（勉强算是提示的线索是该地图查姆·法乌无法出击）。

第四次S中奥拉斩改为消耗EN。同时因为比尔拜因装甲值降低，使其耐久力也得以超越比尔拜因。

---------------------
ビルバイン
---------------------

由于奥拉转换器出现故障正在维修，该机体一度缺席战场。期间被A国夺取，并在地面路线第24话《提坦斯的崛起》中作为艾伦的驾驶机以敌方身份登场。不过由于其HP和装甲与在我方时相同且未经改造，实力并不突出，建议配合精神指令轻松击落。紧接着在次话《超级杰利尔》中，翔于增援时登机，终于正式加入我方阵营。这也意味着翔被迫从改造好的塞拜因或兹瓦士转驾未经改装的比尔拜因，战斗力有所下降。更糟的是，此时若未大幅提升等级，则玛贝尔尚未习得圣战士技能。不过这比后期被迫转乘GP-03S沦为战力废柴的科还是要强得多……


HP值较前作大幅降低，未经改造时稍显吃力。此外前述的塞巴因或兹瓦士获取更早，除射程外几乎全面超越比尔巴因的性能，因此若未培养除翔之外的AB机师，比尔巴因常被闲置仓库的情况并不少见。

第四次S中奥拉斩改为消耗EN，装甲值下降80点，且作为全AB机体共通的削弱调整，奥拉斩改为消耗EN的武器，但即便如此依然强得离谱。同样容易被隐藏机体抢走常规出场机会的特性也未改变。若采用单机突击战术，为节省EN，可考虑把武器改造的重点从光束剑改为灵气剑。奥拉斩的改造费非常贵，以至于可能最多只能使用三台，有时候甚至是两台奥拉战士。走ゴラオン路线的话，可以拿到一台改造过的ライネック，所以是否改造它是个难题。

因为远程武器强，可以让玛贝露驾驶，充分发挥其远攻击优势。

---------------------
バストール
---------------------
巴斯托作为隐藏要素和加拉利亚一起加入。虽然性能较低，但若想将包括加拉利亚在内的三位圣战士全部作为主力使用，就需要对这台机体进行改造。虽说性能较低，但仅是和那些强得犯规的机体相比，于同期加入的波臣和丹拜因仍属优秀，所以此时反而更适合让能使用集中技能的翔或玛贝露驾驶。配合圣战士们的能力，作为全程作战机体完全不成问题。缺点在于宇宙适应为C，不过仅有两话需要在宇宙作战。亦大量作为敌方机体登场。

第四次超级机器人大战S的开场动画中首次采用3D模型登场。奥拉斩改为能量消耗式。造。

---------------------
ゴラオン
---------------------
在加拉利亚加入的路线加入。

比联邦战舰要坚固得多。有地图炮，可以培训チャム的奇迹，以及其他的圣战士系妖精。具有奇迹+地图炮的毁灭能力。宇宙适应是D，和驾驶员平均下来的地形适应仍然是D，好在B队的宇宙地图就只有两话。

加入时有1段改造。第四次S则加入时仅有机体1段改造，武器没有。

---------------------
グラン・ガラン
---------------------
在加拉利亚没有加入的路线加入。相比ゴラオン，虽然少了两名可有可无的无圣战士能力的机师，但是多了两个妖精，在人员配置方面处于压倒性优势。有移动后可使用的远程武器，速通时可以改造。

没有地图武器，以至于奥拉妖精们只能正常升级。还是比联邦战舰要坚固得多，不过和机师的支持能力比起来，机体黯然失色。宇宙适应是D，和驾驶员平均下来的地形适应仍然是D，好在B队的宇宙地图就只有两话。


---------------------
ハイパーガラバ
---------------------
第四次的废弃数据，第四次S中出场，和ハイパーレプラカーン以及ハイパーライネック比起来，バーン的ハイパーガラバ威力不够，因为没有任何气力提升的事件，而且没有奥拉斩。
