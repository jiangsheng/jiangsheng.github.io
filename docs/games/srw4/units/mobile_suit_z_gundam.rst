.. _srw4_unit_methuss_ms_commentBegin:

代号MSA-005。

攻击能力是摆好看的，机体太脆，而且很多敌人有光线护盾，没有用手加减把敌人HP降到10还是不要上去打了。

本作可变性单位大多在人形状态下性能优于飞行状态，例如变形后无法进行盾牌防御或者切防御。因此建议仅在移动时变形，而在攻击时恢复人型态以获得地形加成。例外大概是S高达变形后地形适应更佳，存在变形才能发挥优势的情况。

本作修理能力和机师等级有关，机动战士系的机师等级很容易用地图武器练满，所以超级系的修理机都是摆好看的……

因为限界上限299，终盘生存能力堪忧，建议改造HP和装甲。

.. _srw4_unit_gundam_mk_ii_commentBegin:

编号RX-178。

在初期机动战士中属于运动性强的，而且有重战机系无法防御的导弹武器。因为陆地适应为B，战力甚至弱于钢加农。不过因为可能入手G防卫号合体，所以如果有计划使用超级高达获得双份经验值，则可提前改造以更舒适地推进前期关卡。实用性远超毫无发展空间的高达和GP-01Fb。

因为培养机师所需，缺乏资金时可改造巴尔干炮，资金宽裕时可改造光束步枪用于反击，仅凭地形适应A即可打出和空B的灵格斯相当的伤害。本作陆A机师远多于陆A机体，本机改造价值远高于灵格斯，直到在PS版中灵格斯也改成了空A，本机才黯然失色。在S高达、ZZ高达等强力机体获取之前，将和灵格斯竞争Z高达之后的次级主力地位。

超级高达，正式名称是高达Mk-II防卫号。需要\ :ref:`ガンダム Mk-II <srw4_unit_gundam_mk_ii>`\ (高达 Mk-II)和\ :ref:`Gディフェンサー <srw4_unit_g_defender>`\ (G-防卫号)分别出击，在战场上合体才能出现。

本作虽然开放了副驾驶员的精神，但是获得经验的公式方面，不如上作的按主驾驶员计算，而是分别独立计算，所以不建议继续让低等级驾驶员驾驶高达 Mk-II。尽管如此，如果使用幸运击落高等级敌人，仍能有效加速基斯这样的战斗能力差且有幸运精神的机师升级。问题是本作中地图武器升级更快，甚至可以升到远远超过敌方等级的程度，所以在后期用处不大。在初期升级的时候注意要压级，不要超过敌人等级-8，这样在\ :ref:`百式 <srw4_unit_hyaku_shiki>`\ 入手的时候可以立刻用地图炮刷级，让经验值最大化。

陆地适应是A，实际性能比很多账面数据更高的机体要好，攻防比\ :ref:`ガンダム Mk-II <srw4_unit_gundam_mk_ii>`\  (高达 Mk-II)有所提升。机体能力改造继承ガンダム Mk-II，武器改造继承两个合体的机体的改造段数。

比起ガンダム Mk-II来，超级高达的HP、移动力、地形适应、运动性、装甲、限界都有小幅提升——不过还是很脆，可以加装高性能雷达避免被反击。后期面对敌人时逐渐力不从心，不过这时候应该多用地图炮升级而不是再依赖超级高达的两份经验值。

第四次S中对多数MS的攻击力和地形适应进行了改善，但是超级高达性能并未一起提高，以至于泯然众人矣。虽然双份精神和经验值仍旧有吸引力，但是到能用地图炮刷机的游戏中期，似乎没有必要再挤占出击名额。

代码MSZ-006。奥古和阿纳海姆联合研发的可变性机动战士。地形适应空陆都是A，而且变形之后移动力也比较强，是可以用到最后的机体。大招消耗少，攻击力也不错，改造一下就可以打穿敌人的奥拉光线护盾，威力高过浮游炮。

游戏大多数时间是陆战，这也意味着它比νガンダム（陆B）都要强，在EN回复地形上基本上是本作最强的MS，排除犯规的ZZ不算的话。在机动战士系里实用性只有ビギナ・ギナ可以相比，不过ビギナ・ギナ入手太晚了。

第四次S里武器威力有所提高，不过浮游炮的威力提高之后，相对弱了一些。

尽管可以变形，但是我军空A的机师比较少，而且要利用地形补正的话就不能在空中，所以除了强制以空中形态出击的地图和赶路之外，通常都不会以ウェイブライダー形态作战。
