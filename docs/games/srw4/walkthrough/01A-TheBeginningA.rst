第一话Ａ 発端Ａ
===============================

作为奈梅亨士官学校第二名毕业的高材生，主人公最大的梦想就是加入朗德·贝尔部队。念念不忘这一点的他已经成为了父亲正在开发的机体亡灵的测试驾驶员。加入朗德·贝尔部队之后，他赶在队伍执行约翰·高文中将的绝密任务之前向阿姆罗·丽代理司令官报到。
简单来说，这个任务是调查青木原的一颗陨星，朗德·贝尔怀疑它实际上是飞船，甚至可能是第三次中的敌人，代码为监察者的外星人的飞船。这里速度是重点，因为提坦斯也可能在附近。 为了辅助这个行动，在裁军中被裁减的魔神Z队也加入了行动队伍。

在快到达目标地点时，本队发现魔神Z队已经在和ＤＣ残部战斗了。
