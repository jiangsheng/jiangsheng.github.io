第二十话Ｂ 勇者ライディーン
================================

:doc:`19_baron_ashuras_final_moment` 之后，如果主人公是超级系，那么进入本话。

-------------
评论
-------------

除了シャーキン之外，其他敌人都不能对ライディーン造成实质性伤害。

使用ひびき洸一个人就可以消灭所有敌人杂兵，攒够了气力可以用ゴッドバ－ド秒杀シャーキン，等级相对较高的神宮寺力可以暂时一边凉快去。
