到达菲律宾之后，你发现有两条路可以到达基地，西北方向比较长一点，而西南南方向则有更多的巡逻部队。选择南南西进入\ :doc:`18a_intruder_south_south_west`\ ，路线短一些，但是敌人多，强化芯片也好；选择北西进入\ :doc:`18b_intruder_northwest`\ ，路线长一些，但是敌人少。
